Concrete Mathematics notes

Posted on 2018-04-23 |

Recently I took sometime to read the Concrete Mathematics in preparation for ACM-ICPC. But it turned out to be a quite interesting and useful book which is well edited.
This post will mainly function as a memo in case I suddenly don’t understand some tricky part :P

Chapter1. Recurrent Problems

Lines in the plane

I think this part is really interesting. The origin of this kind of problem is simple:
How many slices of pizza can a person obtain by making n straight cuts with a pizza knife? => What is the maximum number $ L_{n} $ of regions defined by n lines in the plane?
Some variants of this problem can replace straight lines with bent lines or zig-zag lines. The key to this kind of problem is to find the maximum number of new intersections that can be generated between the new line $ l_{n} $ and previous (n-1) lines. New regions are generated along with new intersections.
The closed form of $L_{n}$ is always got by listing out the recurrence. For the straight line case,
$$ L_{0} = 1;L_{n} = L_{n-1}+(n-1)+1 $$
Here (n-1) is the number of new intersections. By observation, (n-1)+1 = n is the number of new regions.
The same rule applies to other scenario where lines are not necessary straight.

Space division

What is the maximum number of three dimensional regions that can be defined by n different planes?
Sol: The number of new 3D regions defined by each new cut is the number of 2D regions defined in the new plane by its intersections with the previous planes.
$$P_{n} = P_{n-1} + L_{n-1} $$
Actually
$ X_{n} = C_n^{0} + C_{n}^{1}; $
$ L_{n} = C_n^{0} + C_{n}^{1}+C_{n}^{2}; $
$ P_{n} = C_n^{0} + C_{n}^{1}+C_{n}^{2}+C_{n}^{3}; $
Really nice pattern.(Sounds Trumpy though…)

The Josephus Problem

Background: N people are standing as a circle and numbered from 1 to n. Every time the third remaining person will be killed until there is less than d=2 people.
The problem: Determine the survivor’s number, J(n).
$J(2^{m}+l) = 2l + 1, for m \geq 0 and 0\leq l < 2^{m}$
Generalization:
$f(j) = \alpha_{j}, for 1 \leq j < d;$
$f(dn+j) = cf(n)+\beta_{j}, for 0 \leq j < d and n \geq 1;$
We start with numbers in radix d and produce values in radix c. That is, it has the radix-changing solution
$f((b_{m} b_{m-1} \ldots b_{1}b_{0})_{d}) = (\alpha_{b_{m}}\beta_{b_{m-1}}\beta_{b_{m-2}} \ldots \beta_{b_{1}}\beta_{b_{0}})_{c} $
For example, if we observe that
$f(1) = 34$
$f(2) = 5$
$f(3n) = 10f(n)+76$
$f(3n+1) = 10f(n)-2$
$f(3n+2) = 10f(n)+8$
Here we have d = 3 and c = 10.
This radix-chanding solution tells us to perform a digit-by-digit replacement from radix d to radix c.
For example $f(19) = f((2,0,1)_{3}) = (5,76,-2)_{10} = 1258$.
